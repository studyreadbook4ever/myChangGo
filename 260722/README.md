# 잠깐 맡김소

지금 당장 처리할 수는 없지만 잊고 싶지도 않은 생각을 잠깐 맡기는, 작은 로컬 생각 보관소입니다. 할 일 목록을 하나 더 만드는 대신 머릿속 점유만 가볍게 내려놓는 앱이 있으면 좋겠다는 생각에서 만들었습니다.

<https://studyreadbook4ever.github.io/myChangGo/260722/>

처음 사용한다면 앱 상단의 `사용법`을 누르거나, [상세 사용 가이드](example.md)를 읽어 보세요.

## 할 수 있는 일

- 떠오른 생각을 짧게 적어 즉시 보관
- 맡긴 생각을 검색하고 필요한 항목만 골라 보기
- 다시 꺼내 볼 시점과 상태를 한눈에 확인
- 완료한 생각을 정리하거나 필요할 때 되돌리기
- JSON 파일로 내보내 백업하고 다른 브라우저에서 가져오기
- 한 번 연 뒤에는 네트워크 없이도 사용
- 작은 화면부터 데스크톱까지 같은 흐름으로 사용

## 개인정보와 저장

생각은 이 브라우저의 로컬 저장소에만 보관됩니다. 계정, 서버 전송, 분석 도구, 광고, 외부 리소스가 없으며 다른 기기와 자동으로 동기화되지 않습니다. 브라우저의 사이트 데이터를 지우면 저장한 생각도 함께 삭제되므로 중요한 내용의 유일한 보관 장소로는 사용하지 마세요.

## 키보드 단축키

| 키 | 동작 |
| --- | --- |
| `N` | 새 생각 입력란으로 이동 |
| `/` | 검색란으로 이동 |
| `Ctrl`/`⌘` + `Enter` | 작성 중인 생각 저장 |
| `Esc` | 열린 대화상자 닫기, 입력란에서는 내용 유지하고 포커스 해제 |

텍스트를 입력 중일 때는 저장과 취소를 제외한 전역 단축키가 입력을 방해하지 않습니다.

## 로컬에서 실행

`260722` 폴더에서 정적 파일 서버를 실행합니다.

```bash
python3 -m http.server 5173
```

그다음 <http://localhost:5173>을 엽니다. 서비스 워커는 `file://` 주소가 아니라 localhost나 HTTPS에서 동작합니다.

## 검사와 테스트

Node.js가 설치되어 있다면 별도 패키지 설치 없이 실행할 수 있습니다.

```bash
npm run check
npm test
```

`npm run check`는 앱과 서비스 워커 JavaScript의 문법을 확인하고, `npm test`는 내장 Node.js 테스트 러너로 테스트를 실행합니다.
